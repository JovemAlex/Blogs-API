<h1>Blogs API</h1>
<p><em>Projeto desenvolvido com os conhecimentos adquirido no módulo de Back-End da <a href="https://www.betrybe.com/" alt="Site da Trybe">Trybe!</a></em></p>

<h3>Objetivo:</h3>
<p>Neste projeto o objetivo era desenvolver uma API e um banco de dados para a produção de conteúdo para um blog! Desenvolvendo uma aplicação em Node.js usando o pacote sequelize para fazer um CRUD de posts.</p>
